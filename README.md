# 🏥 Hospicare – Hospital Management System

Hospicare is a hospital management application designed to improve communication and organization between patients, doctors, and laboratory staff. The system allows patients to access their medical information, manage appointments, and receive laboratory results in one centralized platform.

## 📌 Project Overview

This project was developed to simplify hospital workflows and improve the accessibility of healthcare services. Patients can manage their medical records and appointments, while healthcare professionals can manage prescriptions, reports, and test results.

## ⚙️ Technologies Used

- Java
- XML
- MySQL
- Android Development
- Git / GitHub

## ✨ Features

- Patient portal to view **medical records**
- **Appointment management system**
- Laboratory section to **request tests and view results**
- Doctors can manage **prescriptions and reports**
- Secure access to patient data
- Integrated **hospital location map**

## 🏗 System Modules

### Patient
- View personal medical records
- Book and manage appointments
- Access lab test results
- Upload medical reports

### Doctors
- Review patient information
- Add prescriptions
- Manage appointments

### Laboratory
- Manage test requests
- Upload test results

## 🎯 Project Goal

The goal of this project is to improve healthcare communication and allow patients to easily access their medical information while ensuring proper management of hospital data.
